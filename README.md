## PESTFLY consolidated workflow

This repository provides a unified and application-oriented continuation of the 
analytical workflows above, integrating their contributions into a practical 
diagnostic and operational tool for SNP- and COI-based origin tracing.

*The workflow is under active development. Elements will be made publicly available 
in stages as they become validated, consolidated, and fully documented.*
